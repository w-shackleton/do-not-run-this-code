Installation Help

Installing^

Download the installer from the main DroidPad page, then once it is downloaded, launch it.

The install is quite easy, just accept all default settings.

Up to here:

Click ‘Install’ to install and set up DroidPad.

Once the install has finished, it will launch the setup program.

Configuring^

PPJoy Install^

READ THESE INSTRUCTIONS! Otherwise you may set DroidPad up incorrectly.

This can be run again from the start menu.

Click Next

Click Install to run the installer for PPJoy, the Joystick driver.

You can click Skip if you don’t want to install the driver, or if you already have it installed.

If you do skip, then you can only use your phone as a mouse, not as a Joystick.

Let the installer complete fully. It will then tell you if PPJoy installed properly.

If it didn’t work, try reinstalling it. To skip the installation, click skip.

PPJoy Configuration^

The next step is to configure PPJoy.

In the next window, you can select either an existing controller, or create a new one, either automatically or manually.

If there is an existing controller (“Controller 1″ etc.), then choose that one.

Otherwise, choose New. This will try to automatically set up a new joystick, by automatically clicking the buttons in PPJoy.

If you are finding this is not working, click Manual Setup. Then in the window that appears, click Add.

Then, under the ‘Parallel port’ section, select ‘virtual joysticks’

Then click ‘Add’, and you should see a new joystick in the list.

Found new Hardware Wizard^

When the ‘Found new Hardware Wizard’ appears, select default settings, and let it install normally.

The time that this takes varies, and can take a very long time. Let it finish! In Windows 7, this is just a popup in the notification area.

This process may be different for different versions of Windows.

Once this has finished, click ‘refresh’, and the controller should appear in the drop-down list.

Once you have selected it, click Next.

Click Next to go onto USB setup if Joystick setup worked.

USB Install^

The next part is installing the USB drivers. This step can also be skipped, but USB mode is faster & more reliable than using DroidPad over Wifi.

These drivers are only for T-mobile G1, myTouch 3G, Verizon Droid and Nexus One, but may work with other phones as well. You can download drivers for other phones from the internet if these don’t work.

Click (Re)Install, and then wait until the driver finishes installing.

It can take a long time, but let it finish.

Once it finishes, click Next.

Enable ‘Debug Mode’^

Click Next.

Now, on your Android Phone, Go to Settings->Applications->Development.

Make sure that ‘USB Debugging’ is checked.

Then, click Next.

Now, plug in your phone.

If a ‘Found new hardware wizard‘ pops up, then accept all settings normally, in the same way as earlier.

The time that this takes varies, and can take a very long time. Let it finish!

Click Next.

Once the wizard has finished, click Next again.

USB Configure^

In this next dialogue, you can select your phone.

Click Next.

Once it finishes loading, you should see a list of attached phones.

Click yours, then click OK.

If it worked, it will tell you that the phone is connected.

Then click Next.

Finish^

Click Quit.

The installer will then prompt you to run DroidPad. Click Finish.

Congrats, you are done! DroidPad is ready to go! Fortunately, once you have completed the initial setup, DroidPad is easy to use.

Using DroidPad^

When you first load DroidPad, you will see this screen:

This is what all the parts mean:

  • IP Address: If using Wifi, type the address (numbers) that appear on your phone’s screen.
  • USB: Switches between USB and Wifi mode.
  • Current Mode: Shows the settings selected on your phone.
  • Status: Shows if everything is installed properly.
  • Info Window: Displays info when trying to connect etc.

When opening the phone window, you will see this:

  • Start / Stop: Starts & stops DroidPad
  • Go To Buttons: Opens up the buttons page, when using DroidPad
  • Enable / Disable Wifi: Turns Wifi on or off.
  • Settings: Lets you see the settings from the preferences menu

On your phone, if you go to Menu->Settings, then you can choose out of the following options:

  • Mouse Mode: Control the mouse instead of a Joystick
  • Pointer Mode: (Experimental) point your phone left & right, rather than tilting it, to control movement. For this to work, the phone must be pointing at the computer when it is started.
  • Backwards Mode: Reverse the controls (cable coming out towards computer).

Using with Wifi^

On your phone, click Enable Wifi.

You will see in the status bar when wifi is connected. If it doesn’t connect, then go into the wifi settings page and connect to the network that your phone is on.

When it says that it is connected, there will some text labelled ‘IP:’

Type these four numbers into the IP boxes in DroidPad on your computer.

Once you have typed it in, click Start on your phone.

Then click start on the computer.

If all works OK, then you should now be able to control any game by tilting your phone! Click Go To Buttons to bring up the page of joystick buttons.

Using with USB^

Using USB Mode is a lot easier, once it is set up. just tick USB Mode, then plug in your phone. If you want to select a different phone, then go to File->Choose Device.

Once your device is ready, click start on your phone, then start on the computer.
